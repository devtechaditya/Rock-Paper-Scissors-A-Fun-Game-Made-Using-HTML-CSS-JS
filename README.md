# Rock-Paper-Scissors-A-Fun-Game-Made-Using-HTML-CSS-JS

🪨📄✂️ Rock Paper Scissors Game Website

A simple and interactive **Rock Paper Scissors** web game built using **HTML**, **CSS**, and **JavaScript** 🎮  
Play against the computer directly in your browser with a clean UI and instant results.

The goal of this project is to practice **core frontend fundamentals** such as **DOM manipulation**, **event handling**, **game logic**, and **responsive UI design** — without using any external libraries or frameworks 🚀

---

## ✨ Features

- 🧠 Player vs Computer gameplay  
- 🎲 Random computer choices  
- 🏆 Win, lose, and draw detection  
- 🔄 Play again / reset functionality  
- 📱 Responsive design (desktop & mobile friendly)  
- ⚡ Lightweight and fast (pure JavaScript)  
- 🎯 Simple and intuitive UI  

---

## 🛠️ Tech Stack

- 🌐 **HTML5** – Structure  
- 🎨 **CSS3** – Styling & layout  
- ⚙️ **JavaScript (ES6)** – Game logic & interactivity
